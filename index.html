🔌 Diagram Koneksi Hardware

Koneksi detail antara sensor, ESP32, dan aktuator dalam sistem Smart Greenhouse

INPUT SENSORS

📡 DHT22
GPIO 4 → Temp & Humidity
💧 Soil Moisture Sensor
ADC 34 → Analog Input
💡 LDR Light Sensor
ADC 35 → Light Intensity
🔋 Power Supply
5V 3A → VIN & GND

ESP32

DevKit

• Dual-Core 240MHz
• Wi-Fi Module
• 34 GPIO Pins
• ADC & PWM

OUTPUT ACTUATORS

🔌 Relay Module
GPIO 5 → Digital Output
💦 Water Pump DC 5V
Via Relay → Irrigation
📶 MQTT Broker
Wi-Fi → Cloud Platform
🔔 Notification System
Push → Mobile/Web

📋 Pin Configuration:

DHT22: GPIO 4
Soil Moisture: ADC 34
LDR: ADC 35
Relay/Pump: GPIO 5
Power: VIN (5V), GND
Wi-Fi: Built-in Module
